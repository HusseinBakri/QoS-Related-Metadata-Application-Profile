QoS4-3D Web

QoS-Related Metadata for the 3D Web

Metadata Terms

The terms in the vocabulary presented here pertains to all the elements of the Dublin Core Schema & Europeana Data Model, in addition to custom QoS-Related Metadata pertaining to the usage of 3D Web digital models and environments and speherical media.
